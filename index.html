URL shortener service

by Markus Englund

  • Pass a URL as a parameter and receive a shortened URL as response
  • The shortened URL will redirect you to the URL you passed in

Example

Go to https://url-service-yogaboll.herokuapp.com/new/http://www.aol.com

You will receive a response that looks like this:

{ "short_url": "https://url-service-yogaboll.herokuapp.com/r1rNjcrVg", "original_url": "http://www.aol.com" }

The short url will redirect you to the AOL homepage!
